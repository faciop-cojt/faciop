# faciop-client

- 2020年度COJTで作成しているソフトのフロントエンド.
- Nuxt(typescript)で実装中…
- バックエンド
  - https://github.com/faciop-cojt/faciop-api
- 3Dモデルの試着部分の実装
  - https://github.com/faciop-cojt/faciop-face-detection-sandbox

## 初めにやって

```bash
# 依存パッケージ入れる
$ yarn install

# サーバをlocalhost:3000に
$ yarn dev --port 3000]

# GitHub Pagesへデプロイ
$ npm run generate:gh-pages
$ npm run deploy
```
