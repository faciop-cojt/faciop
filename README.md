# faciop-client

- 2020年度COJTで作成しているソフトのフロントエンド.
- Nuxt(typescript)で実装中…

## 初めにやって

```bash
# 依存パッケージ入れる
$ yarn install

# サーバをlocalhost:3000に
$ yarn dev --port 3000]

# GitHub Pagesへデプロイ
$ npm run generate:gh-pages
$ npm run deploy
```
